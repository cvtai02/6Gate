# 6Gate

Root index for the strict split project.

## Folders

- `app/` - NestJS backend business application.
- `ui/` - Next.js frontend application.
- `api-clients/` - Shared TypeScript DTOs and API client.
- `api-mcp-server/` - Operational tooling and smoke-test server.
- `handoffs/` - Temporary coordination documents.
- `agents/` - AI agent instructions.
