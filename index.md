# 6Gate

Root index for the strict split project.

## Folders

- `app/` - NestJS backend business application.
- `ui/` - Next.js frontend application.
- `handoffs/` - Temporary backend/UI coordination documents; completed handoffs move to `handoffs/archive/`.
- `api-clients/` - Shared TypeScript DTOs and API client.
- `api-mcp-server/` - Operational tooling and smoke-test server.
- `agents-instructions.md` - Root instructions for AI agents.
- `rules.md` - Root project rules.

Historical agent notes live in `agents/`.
