# 6Gate

Root index for the strict split project.

## Folders

- `app/` - NestJS backend application.
- `ui/` - Next.js frontend application.
- `handoffs/` - Temporary backend/UI coordination documents; completed handoffs move to the direction's `archive/` folder.
- `AGENTS.md` - Root instructions for AI agents.
- `CLAUDE.md` - Root instructions for Claude.
- `rules.md` - Root project rules.

`agents-instructions.md` remains as a compatibility pointer to `AGENTS.md`.
